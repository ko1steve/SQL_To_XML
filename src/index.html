SQL 產生器

格式說明

  • 在五大區塊語法第一行放置下列對應的特殊字串,讓程式做辨識擷取SQL語法。
    (1) 前置語法:--#PreSQL
    (2) PreProd前置語法:--#PreProdSQL
    (3) Count語法:--#CountSQL
    (4) 異動前/後語法:--#SelectSQL
    (5) 異動語法:--#MainSQL
    (6) 後置語法:--#PostSQL
  • 統一在 SQL 檔每一段 SQL 指令的前一行放置特殊字串「/*--!*/」,讓程式做辨識截取SQL語法。
  • SQL檔案內容需至少有以下的內容才視為完整的SQL檔案
    (1) DML須至少有Count語法、異動前/後語法、異動語法
    (2) DDL須至少有異動語法
  • DML語法結尾的分號(;)將會被 Dimensions 自動註解化
  • 不得在 DDL 指令內部以外的情境使用 GO, COMMIT 語法。若是有以上語法，則會在 Dimensions 自動被註解化
