SQL 產生器

格式說明

  • 在五大區塊語法第一行放置下列對應的特殊字串,讓程式做辨識擷取SQL指令。
    (1) 前置語法:--#PreSQL
    (2) Count語法:--#CountSQL
    (3) 異動前/後語法:--#SelectSQL
    (4) 異動語法:--#MainSQL
    (5) 後置語法:--#PostSQL
  • 統一在 SQL 檔每一段 SQL 指令的前一行放置特殊字串「/*--!*/」,讓程式做辨識截取SQL指令。
  • SQL檔案內容需至少有以下的內容才視為完整的SQL檔案
    (1) DML須至少有Count語法、異動前/後語法、異動語法
    (2) DDL須至少有異動語法
  • 包含GO語法的指令將會被自動刪除
  • 不得使用commit語法
